# ELENA — Профессиональный сайт

Это простой статический сайт (HTML + CSS). Он подготовлен для размещения на GitHub Pages.

Как опубликовать сайт:
1. Создайте репозиторий на GitHub (например: `YOUR_USERNAME/YOUR_REPO`).
2. На вашем компьютере в папке проекта выполните команды (PowerShell):

```powershell
cd 'C:\Users\Elena\Desktop\me'
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

3. GitHub Actions автоматически запустит workflow и задеплоит сайт на GitHub Pages. Адрес будет `https://YOUR_USERNAME.github.io/YOUR_REPO/`.

Если вы хотите, я могу подготовить всё локально и подсказать точные команды для вставки вашей ссылки на репозиторий.
